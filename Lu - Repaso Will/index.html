Respuesta: HTML (Hypertext Markup Language) es el lenguaje estándar utilizado para crear y estructurar las páginas web. Su función principal en el desarrollo web es: Estructura, Semántica, Contenido, Accesibilidad y Interoperabilidad

Respuesta: Las etiquetas HTML son los elementos básicos que se utilizan para definir la estructura y el contenido de una página web. Sintaxis: Las etiquetas HTML se encierran entre los símbolos < y >. Apertura y cierre: La mayoría de las etiquetas tienen una etiqueta de apertura y una de cierre. Elementos anidados: Las etiquetas HTML pueden estar anidadas unas dentro de otras para crear una estructura jerárquica.

Respuesta: CSS es un lenguaje de hojas de estilo utilizado para controlar la presentación y el aspecto visual de los elementos HTML en una página web. Estilos: CSS permite aplicar estilos a los elementos HTML, como color, fuente, tamaño, espaciado, bordes, etc. Diseño: CSS se utiliza para controlar el diseño y la disposición de los elementos en la página, como posicionamiento, flotación, alineación, etc. Responsividad: CSS permite crear diseños web responsivos que se adaptan a diferentes tamaños de pantalla y dispositivos.

Respuesta: Propiedades de texto: font-family: Establece la fuente del texto. font-size: Controla el tamaño de la letra. color: Define el color del texto. Propiedades de fondo: background-color: Establece el color de fondo de un elemento. background-image: Inserta una imagen de fondo Propiedades de diseño: display: Determina cómo se muestra un elemento (bloque, en línea, flex, grid, etc.). float: Flota un elemento a la izquierda o derecha.

Respuesta: El diseño responsivo (responsive design) es una técnica de diseño web que permite que las páginas se adapten y se vean bien en diferentes dispositivos y tamaños de pantalla, desde computadoras de escritorio hasta dispositivos móviles. Media Queries: Las media queries son reglas CSS que permiten aplicar estilos específicos según las características del dispositivo, como el ancho de la pantalla, la orientación, la resolución, etc. Esto permite crear diseños adaptables.

Respuesta: JavaScript es un lenguaje de programación de alto nivel, interpretado y orientado a objetos, que se utiliza principalmente en el desarrollo web. Se integra en el desarrollo web de la siguiente manera: JavaScript se integra perfectamente con HTML y CSS, lo que permite crear experiencias web dinámicas y atractivas. JavaScript se ha convertido en un lenguaje fundamental para el desarrollo de aplicaciones web modernas, como aplicaciones de una sola página (SPA) y aplicaciones web progresivas (PWA). Con el surgimiento de tecnologías como React Native y NativeScript, JavaScript también se puede utilizar para el desarrollo de aplicaciones móviles nativas para iOS y Android.

Respuesta: La interacción entre JavaScript, HTML y CSS se logra a través del DOM (Document Object Model). El DOM es una representación estructurada y orientada a objetos del documento web, que permite a JavaScript acceder y manipular los elementos de la página.

Respuesta: JavaScript puede validar los campos de un formulario antes de enviarlos al servidor, mejorando la experiencia del usuario. JavaScript puede agregar efectos visuales, como animaciones, transiciones y efectos de hover, para mejorar la apariencia y la interactividad de la página web. JavaScript puede cargar y mostrar contenido de manera dinámica, sin necesidad de recargar toda la página.

Respuesta: Git es un sistema de control de versiones distribuido, gratuito y de código abierto, que se ha convertido en una herramienta fundamental para el desarrollo web y de software en general.

Respuesta: Git permite llevar un registro de los cambios realizados en los archivos de un proyecto a lo largo del tiempo. Git permite crear y gestionar múltiples ramas de desarrollo, lo que facilita el trabajo en paralelo y la experimentación. Git mantiene un registro detallado de todos los cambios realizados en el proyecto, lo que permite rastrear y revertir modificaciones en cualquier momento. Git facilita la colaboración entre desarrolladores, ya que permite que múltiples personas trabajen en el mismo proyecto de manera simultánea.

Respuesta: GitHub es una plataforma de desarrollo y alojamiento de repositorios de código basada en Git. Facilita la colaboración en proyectos web de la siguiente manera: GitHub permite a los desarrolladores crear repositorios remotos para sus proyectos, los cuales pueden ser públicos o privados. Estos repositorios actúan como un punto central de almacenamiento y sincronización del código, facilitando el trabajo en equipo. GitHub permite a los desarrolladores crear copias (forks) de los repositorios de otros usuarios, lo que les permite trabajar en sus propias versiones del proyecto. GitHub proporciona un sistema de seguimiento de problemas (issues), donde los usuarios pueden reportar errores, solicitar nuevas funcionalidades o discutir aspectos del proyecto.
